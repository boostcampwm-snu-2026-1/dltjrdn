# checklist — 현재 작업 / 다음 작업 표시

마인크래프트 스킨 생성기 프로젝트의 진행 상황을 한눈에 보여준다.

## 할 일

1. `PLAN.md`를 읽는다 (1~3주차 체크리스트가 있다).
2. 체크박스 상태(`[ ]` 미완료 / `[x]` 완료)를 파악한다.
3. 실제 파일 상태도 확인한다 — PLAN.md의 체크박스와 실제 코드가 다를 수 있으므로, 핵심 파일 존재 여부를 Glob으로 확인한다:
   - `backend/prompt_skill.py` (① 프롬프트 생성)
   - `backend/sdxl_skin.py` (② SDXL)
   - `backend/mc_skin_generator.py` (③ validate_skin)
   - `backend/main.py` (④ FastAPI)
   - `frontend/` (④ React)
4. 아래 형식으로 보고한다.

## 출력 형식

```
## 📍 지금 하고 있는 일
- (가장 최근 손댄, 아직 안 끝난 항목 1~2개)

## ⏭️ 그다음 할 일
- (바로 이어서 해야 할 항목 2~3개, 우선순위 순)

## ✅ 끝난 것 / ⬜ 남은 것 (주차별 요약)
- 1주차 ①: x/총 개
- 2주차 ②③: x/총 개
- 3주차 ④: x/총 개
```

## 규칙
- PLAN.md 체크박스와 실제 파일이 어긋나면 **실제 파일 기준**으로 말하고, 그 불일치를 짚어준다.
- 추측하지 말고 파일을 실제로 확인한 뒤 보고한다.
- 사용자가 진행을 끝낸 항목이 있으면 PLAN.md 체크박스를 `[x]`로 업데이트할지 물어본다.
